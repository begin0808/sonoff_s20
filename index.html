硬體規格與內建晶片

深入了解 Sonoff S20 的物理與內部電路設計，為您的整合奠定良好基礎。

核心晶片

ESP8266 (Tensilica L106)

內建 2.4GHz WiFi 模組，支援自訂韌體刷寫。

負載規格

10A / 2200W

支援 90V - 250V AC 電壓，可用於大部分中小型家電。

記憶體大小

1MB flash memory

足夠載入完整的 Tasmota 或 ESPHome 系統韌體。

內部引腳 (GPIO)

GPIO0, GPIO12, GPIO13

GPIO0 對應物理按鈕，GPIO12 對應繼電器，GPIO13 對應綠色 LED。

故障排除與 LED 狀態解析

當智慧插座指示燈異常或無法連線時，您可以透過此處獲得快速解答。

LED 狀態燈號對譯

連續快閃兩下 (綠燈)

正在配對模式 (Pairing Mode) 中，等待 App 傳送 WiFi 認證。

常亮無閃爍 (綠燈)

成功連線至路由器並已與 eWeLink 雲端伺服器建立穩定連線。

慢速單次循環閃爍 (綠燈)

正在嘗試連線到您的 WiFi 路由器，請確認熱點名稱及密碼是否正確。

熄滅不亮

插座未通電，或已連上網但當前「電源輸出」為關閉狀態（僅指 Relay LED 狀態）。
